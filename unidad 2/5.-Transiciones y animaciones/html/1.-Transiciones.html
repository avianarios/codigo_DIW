CSS Transitions

They allow to animate a property from the old value to the new value smoothly over time when the value of a CSS property changes

To create a transition effect:

  • Define which property will be the transition based on
  • Assign an initial value for this property
  • Define the duration of the effect
  • Change the value of the selected property to trigger transition

Transitions are controlled by using 4 properties:

  • transition-property: CSS property to be animated
  • transition-duration: in s or ms
  • transition-timing-function: bezier curve or step
  • transition-delay: time before animation starts

There is a shortcut property that sums up the previous properties (in that order): transition

Simple CSS transitions

Using background-color or size and linear transitions

  1. Assign a class describing transformation-property
  2. Assign an event handler that toggles a new class related with what's specified at transition-property

Transform transitions

using transforms (rotate, scale, rotate and scale, and translate)

When click at any ball, three classes are added to it: one for the type of animation (linear an any case), other for the CSS property that will change and how much time animation will last and last one for the CSS property itself (different types of transforms at any case)

Transition-timing function

Controls how animation behaves while is active. Two values: bezier curve or steps

  1. Bezier curve: animation speeds up or slows down during certain amount of time
  2. Steps: transition is divided in steps

Already provided Bezier curves:

  • linear: the same speed from start to end
  • ease: slow start, then fast, then end slowly (this is default)
  • ease-in: slow start
  • ease-out: slow end
  • ease-in-out: slow start and end
  • cubic-bezier(n,n,n,n): customized function. It can be tested at developer tools, clicking on bezier icon at CSS properties if the element has a bezier curve defined or at http://cubic-bezier.com/.

Transitions by order of mouseover: Linear, ease, ease-in, ease-out, ease-in-out, custom bezier 1, custom bezier 2, steps
