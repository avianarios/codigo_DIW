Basic flex therminology

Flex is meant to place elements in one single line, but it allows overflow, meaning that elements not fitting will go to the next line.

Elements are organized as the following image suggests

A container with display: flex; has to be created

In this document some examples are shown. Nevertheless Not every value of every property is shown

column-gap and row-gap have been used everywhere in order to let a gap between elements avoiding using margins or paddings

Properties affecting flex container

flex-direction y flex-wrap

Define el eje principal y controla si los elementos se pueden desbordar

Posibles valores de flex-direction: row, column, row-reverse y column-reverse

Posibles valores de flex-wrap: wrap y nowrap

flex-direction

Los elementos se organizan en una columna inversa (flex-direction: column-reverse). Hay un espacio entre ellos

justify-content

Justify-content alinea objetos en el eje principal que por defecto es el horizontal, pero se puede cambiar

Posibles valores: flex-start, flex-end, center, space-between, space-around y space-evenly

justify-content

align-items

Alinea los elementos en el eje secundario

align-items sólo tiene sentido cuando le damos un tamaño al contenedor y es más grande que los elementos que contiene

Posibles valores: flex-start, flex-end, center, strech y baseline

justify-content y align-items

Los elementos se organizan en una fila (flex-direction: row) se centran los elementos en su eje principal (fila) al centro (justify-content: center) y en su eje secundario (columna) arriba (align-items: start)
