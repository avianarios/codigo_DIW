aligning one particular item

justify-self, align-self, and place-self

The first one aligns a grid item inside a cell along the inline (row) axis. values: start end center stretch

the second one aligns a grid item inside a cell along the block (column) axis . values: start end center stretch baseline. It just takes sense when vertical size of container is bigger than the addition of the size of its elements

the third one is a shortcut to place the item along both axis at the same time

justify-self

Item 1: start, item 2: center, item 3 and 5: end, itemp 4: stretch
