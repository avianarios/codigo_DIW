Basic document flow

Simple float example

clear: left: El siguiente elemento no puede estar flotando a la izquierda del elemento anterior. clear: right: El siguiente elemento no puede estar flotando a la derecha del elemento anterior. clear: both: El siguiente elemento no puede estar flotando ni a la izquierda ni a la derecha del elemento anterior. Esto asegura que el elemento se coloca debajo de todos los elementos flotantes.
