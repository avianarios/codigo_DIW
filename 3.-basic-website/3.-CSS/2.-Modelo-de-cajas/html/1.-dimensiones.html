Dimensiones

Soy una caja. Como todo en HTML. Y tengo anchura y altura. Si no se especifica lo contrario, me expando para ocupar todo el ancho de la pantalla (500px)

Esta caja tiene una anchura de 200px. El texto que haya dentro de ella hace que la caja sea tan alta como se necesite para caber dentro

También se puede especificar un rango mínimo y máximo para la anchura y para la altura Para esta caja es 100-250px y 100-250px. La caja se ajusta al contenido sin salirse de los límites

Tamaño de las cajas

Hay dos opciones a la hora de darle tamaño a las cajas: content-box y border-box

  • content-box: al valor de anchura y altura hay que sumarle borde, relleno y/o margen. Valor por defecto.
  • border-box: el valor de anchura y altura ya incluye el borde, relleno y márgenes.

Esta caja está en modo border-box. Cuando le demos anchura y altura, ya estará incluido borde y relleno

Esta caja está en modo content-box. Cuando le demos anchura y altura, habrá que sumarle borde y relleno
