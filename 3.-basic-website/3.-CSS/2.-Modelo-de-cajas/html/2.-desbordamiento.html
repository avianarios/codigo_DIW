Modelo de cajas

Desbordamiento

Cuando se fija las dimensiones de las cajas ( 200x75px (an x al) en el ejemplo) y el texto no cabe, hay varias opciones

Si el texto no cabe, se desborda (overflow). Este es el comportamiento por defecto (overflow=visible) Hay retornos de línea para que la siguiente caja se pueda leer (mal hecho, no se hace así)







El texto de esta caja no cabe y se va a desbordar. Vamos a usar overflow=hidden para ocultar el contenido que se sale de la caja y que no se vea por fuera.

El texto de esta caja no cabe y se va a desbordar. Vamos a usar overflow=scroll para que aparezcan barras de desplazamiento y poder leer el contenido que se sale de la caja.
